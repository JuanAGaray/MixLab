¿Cómo funciona?

Drinzz ofrece un modelo llave en mano para instalar y operar un punto de granizados dentro de un local comercial (tiendas, droguerías, panaderías, centros de servicio, etc.). La idea es simple: tú tienes el flujo de personas; nosotros ponemos la operación lista para vender.

1

Evalúamos tu local

Revisamos ubicación, flujo de clientes y espacio disponible para el punto.

2

Instalamos todo

Llevamos la infraestructura: máquina, insumos base, montaje y puesta en marcha.

3

Operamos el punto

Asumimos luz, insumos y alquiler del modelo. También colocamos operadores si se requiere.

4

Repartimos utilidades

Empiezas en {{ drinzz_contract.associate_pct_month1 }}/{{ drinzz_contract.operator_pct_month1 }}; si superas {{ drinzz_contract.billing_threshold|pesos_colombianos }} subes a {{ drinzz_contract.associate_pct }}/{{ drinzz_contract.operator_pct }} y puedes sumar +{{ drinzz_contract.maintain_bonus_pct }}%.

¿Qué asume Drinzz?

Para que el asociado no cargue con la operación diaria, Drinzz cubre los gastos principales del punto:

Luz

El consumo eléctrico asociado a la operación del punto de granizados.

Insumos

Bases, vasos, tapas y consumibles necesarios para mantener la venta activa.

Alquiler

Drinzz asume el componente de alquiler dentro del esquema del punto.

Infraestructura

Máquina y montaje del punto listo para vender en tu local comercial.

Operadores

También colocamos personal operador para atender el punto cuando se necesita.
